# IT1214-Student-Grade-System

## Project Purpose

This project is a Java console-based Student Grade Management System developed using Object-Oriented Programming (OOP) concepts.

The system allows users to:

- Add student records
- Display all students
- Search students by ID
- Calculate average marks

## How to Run the Program

1. Compile the Java files:

javac Student.java StudentManagement.java

2. Run the program:

java StudentManagement
